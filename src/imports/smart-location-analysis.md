Add an advanced Smart Location Intelligence section in the YieldX platform after the feasibility study stage. This feature analyzes the market environment based on the business location already entered by the user in earlier steps.

Important:
Do NOT ask the user to re-enter the location. The system must automatically use the previously submitted location data.

---

SMART LOCATION ANALYSIS DASHBOARD

Design an interactive dashboard with the following visual analytics components:

1. Population Density Visualization
   Instead of displaying only numeric values, show a heat map overlay on the map representing population density in the area.

Areas with higher population should appear in warmer colors (red/orange), while lower density areas appear in cooler colors.

This helps users visually identify high-demand zones.

---

2. Competitor Detection Map

Display an interactive map with pin markers showing nearby competitors related to the user's business category.

Each competitor pin should include:
• business name
• rating
• distance from selected location
• small brand/logo icon if available

Users should be able to click on competitors to view details.

Additionally, allow the user to:
• select specific competitors
• select multiple competitors
• select all competitors

These selections will be used for deeper competition analysis.

---

3. Competition Level Indicator

Create a dynamic gauge meter that visually represents the competition intensity in the selected area.

The gauge fills with dynamic color:

Green → Low Competition
Yellow → Moderate Competition
Red → High Competition

The score should be calculated based on:
• number of competitors
• proximity
• market saturation

---

4. Market Activity Score

Design a radial progress visualization showing a score from 0–100 representing the commercial activity of the area.

Include small sparkline trend graphs to illustrate factors such as:
• number of businesses
• customer traffic
• growth trends

Example display:
Market Activity Score: 82 / 100

---

5. Location Suitability AI Meter

Create an AI confidence meter that predicts how suitable the selected location is for the user’s business idea.

The meter should show:

Location Suitability: 74% Match

Below the meter display explainable AI factors, such as:

• Strong population density
• Moderate competitor presence
• High commercial activity
• Good accessibility

These factors help users understand WHY the location is recommended.

---

DESIGN STYLE

Use a modern analytics dashboard design with:

• interactive maps
• data visualization
• smooth animations
• dark theme compatible with the existing YieldX interface
• AI-driven indicators and smart metrics

Ensure all analytics are visually clear, interactive, and easy to interpret.
