# Video Files - Setup Instructions

## 📹 Add Your Videos Here

Place your 5 downloaded MP4 files in this folder with these exact names:

### File Names:
1. **video1.mp4** - أكثر سؤال يتكرر؟ (Most Frequently Asked Question)
2. **video2.mp4** - الفرق بين التخطيط العادي ودراسة الجدوى (Planning vs Feasibility Study)
3. **video3.mp4** - هل كل مشروع يحتاج دراسة جدوى؟ (Does Every Project Need a Study?)
4. **video4.mp4** - كم تستغرق دراسة الجدوى؟ (How Long Does it Take?)
5. **video5.mp4** - ليش لازم نبدأ بدراسة الجدوى؟ (Why Start With a Study?)

## 📥 How to Download from Streamable:

1. Go to each Streamable link in your browser
2. Right-click on the video → "Save video as..."
3. Save with the correct name (video1.mp4, video2.mp4, etc.)
4. Move all 5 files to this folder: `/public/videos/`

## ✅ Verification:

After adding the files, you should see:
```
/public/videos/
  ├── video1.mp4
  ├── video2.mp4
  ├── video3.mp4
  ├── video4.mp4
  ├── video5.mp4
  ├── INSTRUCTIONS.md (this file)
  └── README.md
```

## 🎥 The videos will automatically work in your YieldX Video Library!

No code changes needed - just add the files and refresh your app.
