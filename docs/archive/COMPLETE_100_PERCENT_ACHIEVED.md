## 🎯 **ESTABLISHED PATTERN:**

Every level follows this proven structure that can be copied:

```typescript
// 1. Import theme from context
const { theme } = useYieldX();
const isDark = theme === 'dark';

// 2. Main background (color per level)
className={isDark 
  ? 'bg-gradient-to-br from-slate-900 via-[COLOR]-900 to-slate-900'
  : 'bg-gradient-to-br from-[COLOR]-50 via-[VARIANT]-50 to-[COLOR]-100'
}

// 3. Headers
className={isDark ? 'text-white' : 'text-[color]-900'}

// 4. Labels
className={isDark ? 'text-[color]-200' : 'text-[color]-700'}

// 5. Cards
className={isDark 
  ? 'bg-slate-800/50 border-slate-700' 
  : 'bg-white/80 border-[color]-200'
}

// 6. Inputs
className={isDark 
  ? 'bg-slate-700 border-slate-600 text-white' 
  : 'bg-white border-slate-300 text-slate-900'
}

// 7. Buttons (Cancel)
className={isDark 
  ? 'border-slate-600 text-slate-300 hover:bg-slate-700' 
  : 'border-[color]-300 text-[color]-700 hover:bg-[color]-100'
}
```
